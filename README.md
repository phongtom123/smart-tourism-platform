# C-SA-T Showcase

Public showcase repository for the C-SA-T smart tourism project.

This repository is intended for portfolio and presentation purposes. The full collaboration repository, private credentials, database backups, local payloads, and deployment secrets are not included.

## Overview

C-SA-T includes:

- A .NET MAUI mobile application for visitors.
- An ASP.NET Core backend API.
- A PHP/XAMPP admin portal.
- Geofence, QR access, tour, store, invoice, and device activity workflows.

## Privacy Note

All secrets, local configuration files, database dumps, API keys, keystores, and personal test data should stay out of this public repository.
